Project

Built from the pasted Colab pipeline

The original notebook downloads ATP, Challenger, WTA, and ITF match CSVs, builds per-tour and per-surface XGBoost models, calibrates probabilities, and exposes an ipywidgets prediction UI. This site packages the project for GitHub Pages.

Data

Jeff Sackmann match files from 2017 through 2026, tagged by tour and source.

Features

Elo, surface stats, serve and return rates, fatigue, H2H, rank trend, and slam history.

Models

Separate ATP and WTA models for hard, clay, and grass with probability calibration.

GitHub Pages

Publish in one repo

  1. Create a GitHub repository and add these files to the repository root.
  2. Commit and push to the default branch.
  3. Open Settings, then Pages, then publish from the default branch root.
